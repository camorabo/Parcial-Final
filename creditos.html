Temas y Contenido de la Página

Se usó como fuente, la plantilla de bootswatch Sketchy, que es de donde vienen las imágenes.

Se utilizaron textos en línea otras plantillas de Bootstrap y Javascript para los efectos de las imágenes y la inserción de tablas.

Se procuró usar imágenes que no tienen problemas de copyright en pexels.com.

Varios estilos css se modificaron para esta página web.

El conocimiento de las etiquetas de una página web y los atributos fueron indispensables a la hora de mejorar la presentación de esta página web.

Lo más usado para darle forma a la pñagia web fue la plantilla de Layoutit de Bootstrap.

Las imágenes animadas del equipo de trabajo, propaganda en la página de presentación y le logo se diseñaron en canva.

Varias pestañas implementan Bootstrap pero fue indispensable a la hora de darle la funcionalidad con el LocalStorage como nos solicitó la profesora.
